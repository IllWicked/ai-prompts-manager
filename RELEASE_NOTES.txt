Версия 4.2.12 Hotfix

ЧТО НОВОГО:
- Исправлено ложное срабатывание детекции языка на подстроках (например, «Дания» внутри «создания»)
- Исправлен undo после удаления блока — блок больше не улетает в угол при отмене
- Полная переработка Undo/Redo системы (v2, command-based):
  * Snapshot только по явной команде из точки действия пользователя
  * Save-функции больше не триггерят undo — устранены двойные записи
  * UndoManager API: snapshot, undo, redo, switchTab, renameTab, deleteTab
  * Захват состояния из памяти (не из localStorage) — консистентные снимки
  * Синхронный флаг isRestoring вместо setTimeout-хаков
  * Debounce только для набора текста, force-snapshot для деструктивных операций
  * Autosave больше не создаёт лишних snapshot-записей
