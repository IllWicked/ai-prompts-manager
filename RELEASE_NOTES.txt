Версия 4.3.4

ИСПРАВЛЕНИЯ:
- Убран перехват window.fetch (monkey-patching) — основная вероятная причина блокировок аккаунтов. Мониторинг генерации заменён на DOM-polling (querySelector), подсчёт загрузок — только нативный Rust (WebResourceRequested)
- Убран перехват history.pushState/replaceState — отслеживание URL теперь через popstate + polling каждые 2 сек
- Убрана передача данных через URL (параметры __apm_org, __apm_project в history.replaceState) — org_id читается только из cookie через CDP
- Убран прямой вызов fetch('/api/organizations') для получения org_id — fallback удалён, остался только cookie
- Вставка текста в Claude через ClipboardEvent('paste') вместо editor.commands.insertContent() — идентично реальному Ctrl+V
- Рандомная задержка 300-700мс перед авто-отправкой вместо фиксированных 200мс
- Мониторинг генерации на фоновых табах через Rust CDP polling (работает на суспендированных WebView2 без пробуждения)
- Оффлайн-режим по умолчанию отключен
- Убраны всплывающие подсказки с маркеров языковых переменных
