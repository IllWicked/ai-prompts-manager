Версия 4.4.2

НОВЫЕ ФУНКЦИИ:

- Claude Counter: встроенные индикаторы использования лимитов (session 5h / weekly 7d) с прогресс-барами и обратным отсчётом до сброса. Таймер кэша (5 мин) рядом с заголовком чата. Обновляется автоматически после каждой генерации.

- Knowledge Upload: скачанные MD-файлы автоматически загружаются в knowledge активного проекта Claude. Файл удаляется с диска после загрузки. Уведомление: «📎 file.md → knowledge ✓».

- Автосбор данных из Google: новый тип блока на холсте — скрапер. Задаёте поисковые запросы, приложение само открывает Google, собирает результаты и сохраняет страницы. Собранные файлы загружаются в knowledge проекта Claude — он получает реальные данные вместо встроенного поиска.

- Скрытие дисклеймера «Claude is AI and can make mistakes» из нижней части чата.

- Auto-continue: при достижении tool-use limit Claude автоматически нажимает Continue. Двухступенчатая детекция (кнопка + фраза в последнем сообщении) исключает ложные срабатывания. Включается в Настройки → Дополнительно.

УЛУЧШЕНИЯ:

- Покраска блоков: цветовая маркировка блоков в пайплайнах для визуальной ориентации.

- View mode: масштабирование колёсиком мыши и навигация по холсту кликом средней кнопки.

- Toolbar auto-hide: навигационная панель Claude (← → ⟳ ⬇) скрыта по умолчанию — вместо неё тонкий индикатор внизу. Наведение показывает панель, скрывается через 1 сек.

- Новый гео: Эфиопия (ET).

- Модалка редактирования промпта в view mode стала значительно больше — textarea занимает почти весь экран.

ИСПРАВЛЕНИЯ:

- Множественные загрузки: диалог «Allow multiple downloads?» больше не появляется. Разрешение для claude.ai прописывается автоматически в профиль WebView2 при запуске. Если ранее случайно нажали «Нет» — исправится само при следующем запуске.

- Мелькание чёрного прямоугольника при запуске приложения.

- Вставка текста с прикреплёнными файлами работает корректно.

- Индикатор генерации сбрасывается при навигации на другую страницу.
